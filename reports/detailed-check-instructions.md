# 詳細品質チェック項目

**最終更新**: 2026-02-12（Issue #31の84件修正から得た教訓を反映）

## 1. ハルシネーション・法律的事実誤認（最重要⚠️）

### 1.1 条文の内容

- [ ] 条文の内容を正確に説明しているか
- [ ] 条文の解釈が一般的な通説・判例と矛盾していないか
- [ ] 存在しない条文や条項への言及がないか
- [ ] 法律用語の意味を正確に説明しているか
- [ ] **原文の条文番号と解説の条文番号が一致しているか**
- [ ] **解説が別の条文の内容になっていないか**（条文番号取り違え）

### 1.2 判例・学説

- [ ] 架空の判例を語っていないか
- [ ] 最高裁判例、重要判例を正確に引用しているか
- [ ] 通説と異なる見解を通説として説明していないか

### 1.3 制度の歴史・経緯

- [ ] **法改正の時期を間違えていないか**
- [ ] **改正の理由や背景を正確に説明しているか**
- [ ] **旧法の内容を間違えて説明していないか**
- [ ] **存在しない法改正を語っていないか**
- [ ] **歴史的事実（明治民法、戦後改正など）を正確に説明しているか**
- [ ] **歴史的な年号・日付を正確に記載しているか**（例：東西ドイツ統一=1990年）
- [ ] **条約発効日、法改正日などの重要な日付が正確か**

### 1.4 数字・統計

- [ ] 根拠のない数字や統計を使っていないか
- [ ] 時効期間、金額、期間などを正確に説明しているか

### 1.5 他の条文との関係

- [ ] 引用した他の条文が実在するか
- [ ] 他の条文との関係性を正確に説明しているか
- [ ] 「この条文により」「この規定に基づき」などの説明が正確か

## 2. 文章の長さの異常

- [ ] **極端に短い**: 1-2文、50文字未満
- [ ] **極端に長い**: 不必要な繰り返し、冗長
- [ ] **重複**: 同じことを二回以上言っている

## 3. 大阪弁ペルソナの一貫性

- [ ] **春日歩先生らしさ**: 優しい女性教師の口調か
- [ ] **和歌山弁ベース**: 「〜やで」「〜やん」「〜けど」
- [ ] **禁止表現**: 男性表現（わい、わいら、おんどれ）
- [ ] **商人表現**: 利益、儲け、商売、投資など

## 4. 例え話の品質

- [ ] **登場人物名**: A/B/太郎/花子は良、HHH/XXXなど変な名前はNG
- [ ] **具体性**: 「例えばな、」で始まる具体例があるか
- [ ] **妥当性**: 条文内容と例え話が対応しているか

## 5. 統一感

- [ ] **構成**: 他の条文と同じような構成か（3段落程度）
- [ ] **トーン**: 他の条文と同じような口調か
- [ ] **詳細度**: 他の条文と同程度の詳細さか（300文字以上）

## 6. その他の気になる点

- [ ] 誤字脱字
- [ ] 不自然な日本語
- [ ] 誤解を招く表現
- [ ] 差別的表現

## 問題の重大度分類

### Critical（最重要）🚨

**法律的事実の誤り - 最優先で修正必須**

- **条文番号の取り違え**（原文と解説が別の条文）
- **歴史的事実の誤り**（統一年、改正年、発効日などの誤記）
- **数字の誤り**（時効期間、金額、期間など）
- **条文内容と解説の明確な矛盾**
- **法律用語の意味の誤り**

### High（重大）⚠️

**法律的正確性とペルソナに関わるもの - 即座に修正必須**

- ハルシネーション（存在しない条文・判例・法改正の言及）
- 重大な事実誤認（法制度の歴史、改正経緯の誤り）
- 通説と異なる見解を通説として説明
- 男性表現の使用（わい、わいら、おんどれ）→ペルソナ崩壊

### Medium（中程度）

- 商人表現の使用（利益、儲け、商売、投資）→教育者らしくない
- ペルソナの不一致
- 変な登場人物名（HHH、XXXなど）
- 文章の長さの異常

### Low（軽微）

- 誤字脱字
- 日付表記の不統一
- 軽微な表現の問題
- 統一感の軽微な欠如
