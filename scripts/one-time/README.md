# 使い捨てスクリプト

このディレクトリには、特定の問題を修正するために一度だけ実行するスクリプトが含まれています。

## 特徴

- **使い捨て**: 問題が解決したら二度と使わない
- **特定目的**: 特定の法律や特定の問題に対応
- **履歴保存**: 実行済みでもアーカイブせず、履歴として保持

## 使い方

1. **問題を特定**: 品質チェックツールで問題を発見
2. **スクリプト作成**: 問題を修正するスクリプトを作成
3. **テスト実行**: サンプル条文で動作確認
4. **本番実行**: 対象法律全体に適用
5. **検証**: 修正結果を確認
6. **保存**: このディレクトリに保存（アーカイブしない）

## メンテナンス

- 実行済みスクリプトは削除せず、履歴として保持
- 新しい問題が見つかったら、新しいスクリプトを追加
- スクリプト名には目的が分かる名前を付ける
