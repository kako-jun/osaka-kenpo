# 墾田永年私財法

## 概要

天平15年（743年）5月27日に聖武天皇により発布された法令。新たに開墾した土地の永久私有を認めた画期的な法律。

## 原文（概要）

「今より以後、百姓の新に墾田を造らむは、多少を限らず永に私財と為し、三世一身の法を停止す」

## 背景

- 養老7年（723年）の三世一身法では開墾意欲が不十分
- 人口増加による耕地不足
- 東大寺大仏造営などによる財政負担増大

## 影響

- 公地公民制の実質的崩壊
- 荘園制度の発展
- 貴族・寺社による大規模開墾の促進
- 中世封建制への移行の契機

## 歴史的意義

- 律令制度の転換点
- 私有財産権の確立
- 日本の土地制度史上の重要な画期
