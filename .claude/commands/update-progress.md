# PROGRESS.mdの更新

`/check-progress` の結果を `.claude/PROGRESS.md` に反映する。

## 手順

1. `python3 scripts/tools/check-all-laws-real-status.py` を実行して最新データを取得
2. `.claude/PROGRESS.md` の該当テーブルを最新データで更新
3. 更新日時を記載（`最終更新: YYYY-MM-DD HH:MM`）
4. 状態の判定を更新（100% → ✅ 完成、99%以上 → 🔄 ほぼ完成、等）

## 更新対象セクション

- 「全体サマリー」のテーブル
- 更新日時
- 完了した法律のリスト（新たに完了したものがあれば追加）

## 更新しないセクション

- 更新履歴（手動で追加する場合のみ）
- 品質基準の定義
- コマンド一覧

## 注意事項

- 手動で数字を書かない。スクリプトの出力結果のみを使用する
- 推測や記憶に基づく情報を書かない

## 対になるスキル

進捗の確認には `/check-progress` を使用。
