# トラブルシューティング

このディレクトリには、特定の技術的課題や品質問題を解決するためのガイドが含まれています。

## ファイル一覧

### animation-troubleshooting.md

**アニメーション関連のトラブルシューティング**

UIアニメーションの問題と解決策：

- スクロールアニメーションの不具合
- モバイルでのパフォーマンス問題
- レイアウトシフトの対処法
- デバッグ方法とベストプラクティス

**重要度**: ★★★☆☆（アニメーション実装時の参考資料）

### ui-consistency-checklist.md

**UI一貫性チェックリスト**

UIコンポーネントの一貫性を保つためのチェック項目：

- カラーパレットの統一
- フォントサイズ・スタイルの統一
- スペーシング・レイアウトの統一
- レスポンシブデザインの確認項目
- アクセシビリティチェック

**重要度**: ★★★☆☆（UI実装・レビュー時の確認用）

## 使い方

- アニメーション実装で問題が発生した際は `animation-troubleshooting.md` を参照
- 新しいUIコンポーネントを作成する際は `ui-consistency-checklist.md` で確認
- 新しい問題と解決策を発見した場合は、該当ドキュメントに追記してください
