# コアガイド

このディレクトリには、プロジェクトの中核となるガイドドキュメントが含まれています。

## ファイル一覧

### translation-style-guide.md

**大阪弁翻訳の統合スタイルガイド**

法律条文を大阪弁に翻訳する際の包括的なガイド。以下の内容を含みます：

- 基本方針（親しみやすさ・正確性のバランス）
- 語尾・文体の統一ルール
- 専門用語の翻訳方針
- 春日歩（大阪さん）のキャラクター設定
- 具体例と注意点

**重要度**: ★★★★★（翻訳作業の必須リファレンス）

### law-addition-guide.md

**新しい法律を追加する完全手順書**

新規法律の追加時に必要なすべての手順を網羅：

- データファイル作成手順
- メタデータ設定方法
- ルーティング設定
- 進捗管理への登録
- テスト・確認項目

**重要度**: ★★★★☆（法律追加時の必須リファレンス）

## 使い方

- 大阪弁翻訳を行う際は、必ず `translation-style-guide.md` を参照してください
- 新しい法律を追加する際は、`law-addition-guide.md` に従って作業を進めてください
- これらのガイドは随時更新されるため、作業前に最新版を確認することを推奨します
