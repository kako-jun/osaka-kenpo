## 🛠️ 進捗確認コマンド

### 全体の進捗を確認

```bash
python3 scripts/tools/check-all-laws-real-status.py
```

### 品質問題を確認

```bash
node scripts/tools/check-commentary-quality.cjs
```

結果は `quality-check-report.json` に出力されます。

### 特定の法律の進捗を確認

```bash
# 民法のファイル数をカウント
ls -1 src/data/laws/jp/minpou/*.yaml | grep -v law_metadata | wc -l

# 商法のStage1完成数をカウント
grep -l "originalText:" src/data/laws/jp/shouhou/*.yaml | wc -l
```

## 📋 4段階の定義

| Stage | 項目            | 説明                           |
| ----- | --------------- | ------------------------------ |
| 1     | originalText    | 原文（法律条文そのまま）       |
| 2     | commentary      | 原文の解説（標準語）           |
| 3     | osakaText       | 大阪弁訳（法律条文を大阪弁で） |
| 4     | commentaryOsaka | 大阪弁解説（解説も大阪弁で）   |

**注意**: 削除された条文（`originalText: ['削除']`）はStage1にカウントしない
